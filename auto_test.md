# 嵌入式软件开发任务模板

## 一、软件架构概述
### 核心设计模式
1. **全局设备结构体**：`tMyDev TMyDev` 集中管理所有状态
2. **非阻塞时间驱动状态机**：`get_local_absolute_time()` 超时比较，禁止 delay()
3. **主状态机 + 子状态机**：`soft_state` 枚举分发到各业务模块的子状态机
4. **TLV通信协议**：AA帧头 + 设备类型 + 帧序列 + 消息类型 + TLV数据 + 校验
5. **Flash参数持久化**：阈值、SN、混液标志、基准峰峰值等关键参数掉电保存

### 硬件平台
1. STM32F103RBT6
2. IAR IDE

## 二、驱动接口
### 接口定义
1. PWM输出：PB6 1KHz  PB7 1KHz
2. ADC采集：PB0 ADC1_IN8 PB1 ADC1_IN9


## 三、任务分解
### 1. 工作状态
初始化 空闲 运行
### 2. 初始化状态
1. 配置PWM输出
2. 配置ADC采集
3. 设置默认参数
4. 配置RTT
5. 初始化完成后，定时10s进入空闲状态
### 3. 空闲状态
1. 空闲状态下，定时10s进入运行状态
空闲状态下，定时10s进入运行状态
### 4. 运行状态
1. 运行状态下，定时10s输出PWM信号（PB6）, 50ms后，在一个PWM周期内平均采集20个ADC信号
2. 判断采集到的ADC信号, 是不是一个0～3.3V的方波
3. 如果是，则认为正常，如果不是，则认为异常
4. 关闭PWM信号（PB6）并打开PWM信号（PB7），50ms后，在一个PWM周期内平均采集20个ADC信号
5. 判断采集到的ADC信号, 是不是全部小于0.02v
6. 如果是，则认为正常，如果不是，则认为异常
7. 关闭PWM信号（PB7）并打开PWM信号（PB6），50ms后，在一个PWM周期内平均采集20个ADC信号
8. 采集完成后，定时10s进入空闲状态

# 脚本用户配置文档

## 一、环境要求

| 工具 | 最低版本 | 用途 |
|------|---------|------|
| IAR Embedded Workbench for ARM | 8.x | 编译 |
| SEGGER J-Link | 7.x | 下载 + RTT调试 |
| J-Link 调试器硬件 | V9 | SWD 连接 |

---

## 二、脚本列表

```
scripts/
├── build.bat          编译工程, 输出 HEX/BIN
├── download.bat       通过 J-Link 烧录到芯片
├── verify_rtt.bat     捕获 RTT 输出用于验收
└── auto_all.bat       一键: 编译 → 下载 → 验收
```

---

## 三、使用方法

### 3.1 编译
```
cd scripts
build.bat
```

### 3.2 下载
```
download.bat
```
前置条件: 先执行 `build.bat` 生成 HEX 文件。

### 3.3 RTT 验收
```
verify_rtt.bat 10                    捕获10秒
verify_rtt.bat 30 0x20000400         指定RTT地址, 捕获30秒
```
输出文件: 项目根目录下 `rtt_output.txt`

### 3.4 一键全流程
```
auto_all.bat                         默认捕获10秒
auto_all.bat 30                      捕获30秒
```

## 四、配置说明

> 换项目或换芯片时, 只需修改每个脚本顶部的 **用户配置区**。

### 4.1 build.bat 配置项

打开 `build.bat`, 找到文件开头的配置区:

```bat
REM ---- 用户配置区 (换项目只改这里) ----
set "IAR_HOME=C:\Program Files (x86)\IAR Systems\Embedded Workbench 8.3"
set "PROJECT_DIR=%~dp0.."
set "PROJECT_NAME=auto_test"
set "BUILD_CONFIG=Debug"
REM ---- 配置区结束 ----
```

| 变量 | 说明 | 示例 |
|------|------|------|
| `IAR_HOME` | IAR 安装路径 | `C:\Program Files (x86)\IAR Systems\Embedded Workbench 8.3` |
| `PROJECT_DIR` | 项目根目录 | `%~dp0..` (默认: scripts的上级目录) |
| `PROJECT_NAME` | 工程文件名 (不含.ewp) | `auto_test` |
| `BUILD_CONFIG` | 编译配置名 | `Debug` 或 `Release` |

### 4.2 download.bat 配置项

```bat
REM ---- 用户配置区 (换项目/芯片只改这里) ----
set "JLINK_HOME=C:\Program Files\SEGGER\JLink"
set "DEVICE=STM32F103RB"
set "INTERFACE=SWD"
set "SPEED=4000"
set "PROJECT_DIR=%~dp0.."
set "HEX_FILE=%PROJECT_DIR%Debug\Exe\auto_test.hex"
REM ---- 配置区结束 ----
```

| 变量 | 说明 | 常用值 |
|------|------|--------|
| `JLINK_HOME` | J-Link 安装路径 | `C:\Program Files\SEGGER\JLink` |
| `DEVICE` | 芯片型号 (J-Link 设备名) | 见下方芯片列表 |
| `INTERFACE` | 调试接口 | `SWD` 或 `JTAG` |
| `SPEED` | 下载速度 (kHz) | `4000` (4MHz) |
| `HEX_FILE` | HEX 文件路径 | `%PROJECT_DIR%Debug\Exe\工程名.hex` |

#### 常用芯片 DEVICE 名称

| 芯片 | DEVICE 值 |
|------|-----------|
| STM32F103C8T6 | `STM32F103C8` |
| STM32F103RBT6 | `STM32F103RB` |
| STM32F103RCT6 | `STM32F103RC` |
| STM32F103VET6 | `STM32F103VE` |
| STM32F103ZET6 | `STM32F103ZE` |
| STM32F030C8T6 | `STM32F030C8` |
| STM32F407VET6 | `STM32F407VE` |
| STM32F407ZGT6 | `STM32F407ZG` |
| STM32G070CBT6 | `STM32G070CB` |
| STM32H750VBT6 | `STM32H750VB` |

> 完整列表见 J-Link Commander 输入 `?` 查看, 或访问 https://www.segger.com/supported-devices/

### 4.3 verify_rtt.bat 配置项

```bat
REM ---- 用户配置区 ----
set "JLINK_HOME=C:\Program Files\SEGGER\JLink"
set "DEVICE=STM32F103RB"
set "INTERFACE=SWD"
set "SPEED=4000"
set "PROJECT_DIR=%~dp0.."
set "RTT_LOG=%PROJECT_DIR%rtt_output.txt"
set "CAPTURE_TIME=10"
set "RTT_ADDR="
REM ---- 配置区结束 ----
```

| 变量 | 说明 | 备注 |
|------|------|------|
| `JLINK_HOME` | J-Link 安装路径 | 同 download.bat |
| `DEVICE` | 芯片型号 | 同 download.bat |
| `RTT_LOG` | RTT 输出文件路径 | 默认在项目根目录 |
| `CAPTURE_TIME` | 默认捕获时长 (秒) | 命令行参数可覆盖 |
| `RTT_ADDR` | RTT 控制块地址 | 留空=自动搜索; 填地址如 `0x20000400` |

#### 如何查找 RTT_ADDR

编译后在 map 文件中搜索:

```
grep "_RTT" Debug\List\auto_test.map
```

输出示例:
```
_RTT    0x2000'0400   0x44  Data  Lc  SEGGER_RTT.o [1]
```

取地址 `0x20000400` 填入配置即可。

## 五、新项目配置示例

假设新项目 `my_project` 使用 STM32F407VET6:

### 步骤 1: 复制脚本
```
my_project/
├── scripts/          ← 从模板复制整个 scripts 文件夹
├── CMSIS/
├── Core/
└── my_project.ewp
```

### 步骤 2: 修改 build.bat
```bat
set "PROJECT_NAME=my_project"
```

### 步骤 3: 修改 download.bat
```bat
set "DEVICE=STM32F103VE"
set "HEX_FILE=%PROJECT_DIR%Debug\Exe\my_project.hex"
```

### 步骤 4: 修改 verify_rtt.bat
```bat
set "DEVICE=STM32F103VE"
```

### 步骤 5: 执行
```
cd scripts
auto_all.bat
```

## 六、故障排查

### 编译失败
- 确认 `IAR_HOME` 路径正确, 末尾不要有反斜杠
- 确认 `PROJECT_NAME` 与 .ewp 文件名一致

### 下载失败
| 现象 | 原因 | 解决 |
|------|------|------|
| `Could not connect` | J-Link 未接/驱动未装 | 检查 USB, 确认设备管理器识别 |
| `Could not find device` | DEVICE 名称错误 | 对照 J-Link 设备列表修正 |
| `Flash download failed` | 芯片写保护或接线问题 | 检查 SWDIO/SWCLK/GND 三线 |

### RTT 无输出
| 现象 | 原因 | 解决 |
|------|------|------|
| `No RTT control block found` | 未指定地址且自动搜索失败 | 在 map 文件找 `_RTT` 地址, 填入 `RTT_ADDR` |
| 输出乱码 | 波特率不匹配 | 确认程序调用了 `SEGGER_RTT_Init()` |
| 程序卡死无输出 | 时钟初始化失败(HSE无晶振) | 检查硬件有无外接晶振 |
